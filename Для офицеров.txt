🛠 ИНСТРУКЦИЯ ДЛЯ АДМИНОВ / ОФИЦЕРОВ

📌 Управление составом:

Добавить игрока:
➡ /add ник

Исключить игрока из состава:
➡ /кик ник

Показать список игроков:
➡ /players

📌 Управление учетом:

Принудительно начать учет сегодня:
➡ /начатьучет

Очистить сегодняшний лист (сброс тестов):
➡ /cleartoday

Создать лист по дате:
➡ /createday дд.мм.гггг

Пересоздать лист по дате:
➡ /clearday дд.мм.гггг

📌 Ручные отметки игроков:

Отметить присутствие игрока:
➡ /adminhere ник

Отметить отсутствие игрока:
➡ /admincant ник причина

Пример:
/admincant Player работа

📌 Проверка активности:

Кто не отметился сегодня:
➡ /когонет

Принудительно отправить очередь данных в таблицу:
➡ /flush

📌 Важно:

* Используйте /cleartoday только для тестов.
* /кик удаляет игрока из состава на будущие дни.
* Все команды доступны только администрации.
